    **computeCurrentCycle(statementDay: number, nowISO: string): BillingCycle:**

    Boundary convention: `boundary_UTC = Date.UTC(year, month0, statementDay, 17, 0, 0)`.
    This equals midnight (00:00) of `statementDay + 1` in Vietnam time (UTC+7).

    Logic:
    1. Construct `vnDate = new TZDate(nowISO, 'Asia/Ho_Chi_Minh')` to get VN year/month/day.
    2. Get `vnDay = vnDate.getDate()`.
    3. If `vnDay <= statementDay`: cycle started LAST month.
       - `prevM = m === 0 ? 11 : m - 1`; `prevY = m === 0 ? y - 1 : y`
       - `startUTC = new Date(Date.UTC(prevY, prevM, statementDay, 17, 0, 0))`
       - `endUTC = new Date(Date.UTC(y, m, statementDay, 17, 0, 0))`
    4. If `vnDay > statementDay`: cycle started THIS month.
       - `nextM = m === 11 ? 0 : m + 1`; `nextY = m === 11 ? y + 1 : y`
       - `startUTC = new Date(Date.UTC(y, m, statementDay, 17, 0, 0))`
       - `endUTC = new Date(Date.UTC(nextY, nextM, statementDay, 17, 0, 0))`
    5. `startVN = new TZDate(startUTC.toISOString(), 'Asia/Ho_Chi_Minh')`
    6. `endVN = new TZDate(endUTC.toISOString(), 'Asia/Ho_Chi_Minh')`
    7. Display dates:
       - `startDisplay = format(startVN, 'dd/MM/yyyy')` — startVN is already midnight of day after statementDay
       - `endDisplay = format(subDays(endVN, 1), 'dd/MM/yyyy')` — subtract 1 day to get last inclusive VN day
       - `statementDateDisplay = endDisplay`
    8. `daysUntilClose = Math.ceil((endUTC.getTime() - new Date(nowISO).getTime()) / 86_400_000)`

    **groupTransactionsByCycle(transactions: CreditCardTransaction[], currentCycle: BillingCycle): BillingCycleGroupData[]:**

    Use a `Map<string, BillingCycleGroupData>`. For each tx:
    - If `tx.billingCycleStart && tx.billingCycleEnd` → use them as key/start/end
    - Else (pending) → use `currentCycle.startISO` / `currentCycle.endISO` as key/start/end
    - Set `isCurrentCycle = cycleKey === currentCycle.startISO`
    Push tx into group's transactions array.
    Return `Array.from(groups.values()).sort((a, b) => b.cycleKey.localeCompare(a.cycleKey))` (newest first).
