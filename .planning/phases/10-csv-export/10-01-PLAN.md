## CSV Requirements
| Column | Source | Format |
|--------|--------|--------|
| Ngày | transactionDate | dd/MM/yyyy (formatDisplayDate) |
| Mô tả | description | string as-is |
| Số tiền | Math.abs(amount) | đ X.XXX format (formatVND) |
| Loại | type + (income\|expense) | "Thu nhập" or "Chi tiêu" |
| Tài khoản | account.accountName or card.cardName | string or "N/A" |
| Danh mục | getEffectiveCategory() override | category string |

## Filter Handling
Export must pass through all active filters via TransactionFilters:
- dateFrom, dateTo (date range)
- txType (income/expense)
- category (overridable)
- accountId (for bank transactions) OR cardId (for credit card transactions)
- search (optional merchant/description search)

Service signature mirrors getTransactions(accountId, filters) pattern but with NO pagination (no cursor, no limit).

## BOM and Encoding
- UTF-8 BOM: `\ufeff` prepended to CSV string before Blob
- Blob type: 'text/csv;charset=utf-8'
- Papa Parse: unparse() with { header: true, delimiter: ',', newline: '\r\n' }
