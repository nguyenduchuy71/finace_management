Create `src/hooks/useBudgetAlerts.ts`:

**Pure function (exported for testing):**
```typescript
export interface BudgetAlert {
  category: Category
  percent: number
  level: 'warning' | 'overbudget'
  spent: number
  budget: number
}

export function computeBudgetAlerts(
  items: Array<{ category: Category; spent: number; budget: number }>
): BudgetAlert[]
```
- Filter items where budget > 0 AND percent >= 80
- For each: compute percent = Math.round((spent / budget) * 100)
- level = percent >= 100 ? 'overbudget' : 'warning'
- Sort descending by percent (worst alerts first)
- Return BudgetAlert[]

**Hook (uses pure function + fires toasts):**
```typescript
export function useBudgetAlerts(
  categoryBreakdown: CategoryBreakdownItem[],
  budgets: Record<Category, number>
): { alerts: BudgetAlert[]; warningCount: number; overbudgetCount: number }
```
- Build items array: map over budgets keys, lookup spent from categoryBreakdown
- Call computeBudgetAlerts(items) via useMemo
- useEffect (fires on mount only — empty dep array or use useRef flag to prevent re-fire):
  - For each alert: if level === 'overbudget', call toast.error with Vietnamese message like "An uong: vuot ngan sach (120%)"
  - For each alert: if level === 'warning', call toast.warning with Vietnamese message like "An uong: gan dat ngan sach (90%)"
  - Use a useRef(false) flag `hasFired` to ensure toasts fire at most once per mount. Set true after first fire.
- Return { alerts, warningCount: alerts.filter(a => a.level === 'warning').length, overbudgetCount: alerts.filter(a => a.level === 'overbudget').length }

**Toast messages (Vietnamese):**
- Warning: `"{category}: sap dat gioi han ngan sach ({percent}%)"`
- Overbudget: `"{category}: da vuot ngan sach ({percent}%)"`

Import: toast from 'sonner', useBudgetStore, CategoryBreakdownItem type, Category type, useMemo, useEffect, useRef.
