  clearBudget: (category: Category) => void

- Guard: if budget === 0, return null (no budget set → hidden, not rendered)
- Calculation: percent = Math.min(Math.round((spent / budget) * 100), 100) — capped at 100 for visual consistency
- Color logic:
  - isOverBudget = percent >= 100 → 'bg-red-600 dark:bg-red-500'
  - isWarning = percent >= 80 && percent < 100 → 'bg-yellow-500 dark:bg-yellow-400'
  - else → 'bg-emerald-600 dark:bg-emerald-500'
- Layout:
  - Flex row: category label | percent label
